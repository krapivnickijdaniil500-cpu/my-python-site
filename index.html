Python 20: Master Guide

От основ синтаксиса до архитектуры приложений

01. Динамические переменные

Python использует ссылки на объекты. Тип данных определяется в момент присваивания значения.

Типы: int (целые), float (дробные), str (строки), bool (логика).
x = 10          # int
name = "Dev"    # str
is_ok = True    # bool

02. Форматирование (f-strings)

Самый быстрый и удобный способ вставлять переменные прямо в текст через фигурные скобки.

Фишка: Можно выполнять вычисления прямо внутри {}.
price = 100
print(f"Итого: {price * 0.9} руб.")

03. Списки (Lists)

Упорядоченные коллекции элементов. Индексация начинается с нуля.

Методы: append (добавить), pop (удалить), sort (сортировка).
users = ["Ivan", "Anna"]
users.append("Max")
print(users[1]) # Anna

04. Логические условия

Операторы if, elif и else управляют потоком выполнения программы.

Важно: В Python отступы (4 пробела) определяют блоки кода.
if age >= 18:
    print("Доступ открыт")
else:
    print("Рано")

05. Итерация (Цикл For)

Используется для перебора списков, строк или диапазонов чисел.

Range: range(5) создаст числа от 0 до 4.
for i in range(3):
    print(f"Шаг №{i}")

06. Словари (Dict)

Хранение данных в формате Ключ: Значение. Работает молниеносно.

Ключи: Должны быть уникальными и неизменяемыми.
car = {"brand": "Tesla", "year": 2023}
print(car["brand"])

07. Функции (def)

Используются для группировки кода, который нужно выполнять многократно.

Return: Возвращает результат работы функции обратно.
def square(n):
    return n * n
res = square(5) # 25

08. Кортежи (Tuples)

Похожи на списки, но их нельзя изменить после создания (Immutable).

Зачем: Работают быстрее списков и защищают данные.
point = (10, 20)
# point[0] = 5 # Вызовет ошибку!

09. Множества (Sets)

Коллекции, содержащие только уникальные элементы в случайном порядке.

Применение: Удаление дубликатов из списка за один шаг.
nums = {1, 2, 2, 3}
print(nums) # {1, 2, 3}

10. Исключения (Try/Except)

Позволяют программе не "падать" при возникновении ошибок (например, делении на 0).

Блок finally: Выполнится в любом случае (даже при ошибке).
try:
    val = 10 / 0
except ZeroDivisionError:
    print("Ошибка деления!")

11. Срезы (Slicing)

Мощный инструмент получения частей строк или списков.

s = "Python"
print(s[:2])  # Py
print(s[::-1]) # nohtyP

12. Ламбда-функции

Анонимные функции в одну строку для простых операций.

add = lambda a, b: a + b
print(add(5, 3)) # 8

13. List Comprehensions

Создание списков в одну строку кода (Pythonic way).

sq = [x*x for x in range(5)]
# [0, 1, 4, 9, 16]

14. Конструкция With

Гарантирует закрытие файлов или ресурсов после работы.

with open("file.txt", "w") as f:
    f.write("Hello")

15. Классы (ООП)

Создание собственных объектов со свойствами и методами.

class Cat:
    def meow(self):
        print("Мяу!")

my_cat = Cat()

16. Импорт библиотек

Использование готового кода от других разработчиков.

import math
print(math.sqrt(16)) # 4.0

17. Map и Filter

Функции высшего порядка для обработки коллекций данных.

nums = [1, 2, 3]
new = list(map(lambda x: x*2, nums))

18. Гибкие аргументы

Позволяют функции принимать любое количество параметров.

def func(*args):
    return sum(args)

19. Декораторы

Функции, которые оборачивают другой код, добавляя ему логику.

@my_timer
def fast_func():
    pass

20. Управление пакетами

Использование PIP для установки внешних модулей (Requests, Pandas).

# В терминале:
pip install requests

🧮 Интерактивный калькулятор

Слева — интерфейс (HTML), справа — логика обработки (Python)

🐍 Исходный код (logic.py)


print("--- Мини-калькулятор ---")

# Запрашиваем данные у пользователя
a = float(input("Введите первое число: "))
operation = input("Выберите действие (+, -, *, /): ")
b = float(input("Введите второе число: "))

# Логика вычислений
if operation == '+':
    result = a + b
    print(f"Результат: {a} + {b} = {result}")
elif operation == '-':
    result = a - b
    print(f"Результат: {a} - {b} = {result}")
elif operation == '*':
    result = a * b
    print(f"Результат: {a} * {b} = {result}")
elif operation == '/':
    # Проверка деления на ноль
    if b != 0:
        result = a / b
        print(f"Результат: {a} / {b} = {result}")
    else:
        print("Ошибка")
print("------------------------")
