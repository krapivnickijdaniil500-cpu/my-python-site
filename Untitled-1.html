01. Переменные (Variables)

Объекты в памяти, на которые ссылаются имена. Тип определяется динамически.

  • Автоматическое управление памятью
  • Snake_case для именования
  • f-строки для быстрого вывода
name = "Emerald"
ver = 3.12
print(f"{name} v{ver}")

02. Списки (Lists)

Динамические упорядоченные массивы. Позволяют изменять данные.

  • Метод .append() для добавления
  • Метод .pop() для удаления
  • Срезы: [start:stop:step]
data = [1, 2, 3]
data.append(4)
subset = data[0:2]

03. Условия (Logic)

Операторы ветвления для принятия решений в зависимости от результата выражения.

  • if / elif / else
  • Логические and / or / not
  • Тернарный оператор: A if C else B
if score > 90:
    res = "Win"
else:
    res = "Try again"

04. Цикл For

Используется для перебора последовательностей (списков, строк, словарей).

  • Функция range() для счета
  • Функция enumerate() для индексов
  • Поддерживает break и continue
for i in range(5):
    print(i * 10)

05. Словари (Dicts)

Коллекции пар "ключ-значение". Оптимизированы для мгновенного поиска.

  • Ключи всегда уникальны
  • Метод .get() против ошибок
  • Метод .update() для слияния
user = {"id": 1, "val": "ok"}
val = user.get("key", 0)

06. Функции (Def)

Блоки кода для многократного использования с параметрами и возвратом.

  • Аргументы по умолчанию
  • *args и **kwargs
  • Документирование через docstrings
def calc(x, y=1):
    return x * y

07. Исключения (Try)

Механизм перехвата ошибок для предотвращения аварийной остановки.

  • Блок try/except
  • Блок finally для очистки
  • Вызов ошибок через raise
try:
    num = 10 / 0
except ZeroDivisionError:
    print("Oops!")

08. Работа с файлами

Чтение и запись данных на диск с использованием менеджеров контекста.

  • Оператор with
  • Режимы 'r', 'w', 'a'
  • Авто-закрытие файлов
with open("f.txt", "w") as f:
    f.write("Python Emerald")

09. Модули (Modules)

Подключение встроенных и сторонних библиотек через пакетный менеджер PIP.

  • Стандартная библиотека (math, os)
  • Установка через pip install
  • Создание своих пакетов
import math
res = math.sqrt(64)

10. ООП (Classes)

Парадигма программирования на основе объектов и классов.

  • Инкапсуляция данных
  • Наследование свойств
  • Метод __init__ (конструктор)
class Bot:
    def __init__(self, n):
        self.name = n
